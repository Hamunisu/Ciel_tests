1. 国内線　便名追加(便名変更予定)
----------------------------------
        
        コピペ推奨:

            {
                "arrival": ["","",""],
                "flightNumber": ""
            },


        例:

            {
                "arrival":["rjcm","mmb","女満別"],
                "flightNumber":"1552\n1554\n1556"
            },

    
    ==解説==

        1. d_fs/の中の編集したい空港のjsonファイルを選択(空港名は出発空港)


        2. コピペ推奨を張り付ける


        3. "arrival": ["","",""],は4レター、3レター、空港名の順で入力


        4. "flightNumber"は便名を入力　(便名が2つ以上ある場合は1つの区切りとして"\(バックスラッシュ)n"を入力)
        
            ※ ファイルの一番下の便名は最後のカンマ/コンマは消してください


===============================================================================================================================================================================================


2. Galleryに写真を追加する手順
-------------------------------

        コピペ推奨:

            <div class="image-item">
                 <img src="../image/gallery/" alt="">
                 <p></p>
            </div>


        例:

            <div class="image-item">
                <img src="../image/gallery/MSFS_HP_oo.png" alt="ハリーポッター伊丹">
                <p>ハリーポッタージェット in大阪国際空港</p>
            </div>


    ==解説==

        1. image/galleryの中に掲載したい写真を保存する　(16:9を推奨)
        

        2. sub_html/gallery.htmlを選択し、<!-- Gallery -->までスクロール

        
        3. src="../image/gallery/"の後に追加した写真名を追加
        

        4. alt=""はエラーで写真が出なかった場合に表示されるテキスト
        

        5. <p></p>は写真の下部に表示されるテキスト(簡単な説明でOK)






    -ほかの手順が知りたい場合はsunaまで連絡をいただければ対応いたします-
